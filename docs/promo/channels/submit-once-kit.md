## 5. 素材生产(只录一次,全渠道复用)

### 5.1 录屏设置

**首选 OBS(免费,能录系统声音——游戏的程序化音乐值得收进视频)**:

- 来源:`macOS Screen Capture`(OBS 30+ 自带系统音频采集;若你的 OBS 版本没有系统声音,先升级 OBS,而不是装虚拟声卡)。采集 Chrome 游戏窗口。
- 设置 → 视频:Base(画布)与 Output(缩放)都设 `2560x1440`(机器吃力就 `1920x1080`);FPS `60`。
- 设置 → 输出 → 录像:格式 `MKV`(防崩溃丢文件,录完用 OBS 菜单 File → Remux Recordings 转成 mp4);编码器 `Apple VT H264 硬件编码`;码率 `40000 Kbps`(1080p 用 25000)。
- 音频:确认"桌面音频"电平在动;麦克风轨**关掉**(纯 gameplay,不收环境音,防止收进孩子的声音)。
- 浏览器全屏(Chrome 按 F11 或 macOS 绿色按钮全屏),游戏画质设置调到高、开 bloom,录之前隐藏鼠标不必要的移动。

**QuickTime 备选**(不推荐,仅当 OBS 装不了):文件 → 新建屏幕录制。注意 QuickTime **录不了系统声音**(只有麦克风),成片没游戏音乐,需要后期在剪映里配。能用 OBS 就用 OBS。

**竖屏 Shorts 的取景**:统一录 2560x1440 横屏母带,剪映里新建 1080x1920 竖屏项目,把母带放大、逐镜头跟随主体裁切(1440p 母带竖裁后有效宽度约 810px,放大到 1080 宽画质可接受)。不要单独再录一遍竖屏——一次录制,两种成片。

- **A2 甲板加炮全流程**:进港 → 商店 → 按 F 装炮 → 出港齐射。喂 Short 1、截图 06。
- **A3 舰上起飞 + 扫射 + flyby**:起飞、攻击、低空掠过自己战列舰各来一遍。喂 Short 2、hero 片、截图 03。
